ADAS Alert System & Road Safety Analytics - Intel View on GitHub

This project analyzed Intel's real-world vehicle data to understand how drivers respond to Advanced Driver Assistance System (ADAS) alerts like forward collision warnings. I built statistical models to test whether alerts caused drivers to slow down consistently and identified key differences in how various vehicle types reacted. The results were visualized in interactive dashboards to provide clear safety insights, and the findings were presented at a national conference.

2022

Role-Based Access Control (RBAC) Web Application View on GitHub

I developed a full-stack web application from the ground up to automate user access management. The system implements a secure Role-Based Access Control (RBAC) model with three distinct user roles (Employee, Manager, Admin) and a complete workflow for access requests. I handled the full technology stack, building the backend logic with Java Servlets, managing data with PostgreSQL, creating the front-end with JSP, and deploying the application on Apache Tomcat. This project demonstrates the ability to build a secure, functional, and end-to-end software solution for a common business process.

2025

Currency Exchange Microservice System View on GitHub

Developed a cloud-native microservices application to demonstrate modern distributed systems architecture. I built three decoupled services—a Eureka Naming Server for service discovery, a Forex Service to provide currency exchange rates, and a Currency Conversion Service for calculating conversions—using Spring Boot 3. The services communicate via REST APIs, implementing core principles of microservices like loose coupling and independent deployability. This project showcases practical hands-on experience with a foundational microservices stack.
